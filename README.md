***mmdeeplearning是一个学习深度学习框架的demo项目，通过该项目了解深度学习的基本理念和各个深度学习框架的使用方式和原理***
- numpy 高纬度数据模型
- matplotlib 绘图组件
- pandas
- mxnet 深度学习框架
- tensorflow 深度学习框架
